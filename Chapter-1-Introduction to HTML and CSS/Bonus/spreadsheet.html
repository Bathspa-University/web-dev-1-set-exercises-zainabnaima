HTML & CSS Quick-Reference Guide

HTML Tags Description Example Usage
<html> Defines the root of an HTML document. <!DOCTYPE html> Page Titles
<head> Contains information about the HTML document. <head></head> Metadata
<body> Contains the content of the HTML document. <body></body> Page content
<h1> to <h6> Heading Levels 1 to 6, with h1 being the most important. <h1>Welcome to My Website</h1> Headers
<p> Defines a paragraph. <p>This is a paragraph.</p> Text content
<a> Defines a hyperlink. <a href="https://www.example.com">Visit Example Website</a> Links
<img> Defines an image. <img src="image.jpg" alt="Description of image"> Images
<div> Defines a division. <div></div> Layout
<span> Defines a span. <p>This is <span>highlighted</span> text.</p> Text styling
CSS Properties Description Example Usage
color Sets the color of text. p { color: blue; } Text styling
font-size Sets the font size. h1 { font-size: 24px; } Text styling
background-color Sets the background color. body { background-color: #f0f0f0; } Layout
padding Sets the padding around an element. div { padding: 10px; } Layout
margin Sets the margin around an element. p { margin: 20px; } Layout
text-align Sets the horizontal alignment of text. h1 { text-align: center; } Text styling
border Sets the border around an element. div { border: 1px solid black; } Layout
font-family Sets the font family. body { font-family: Arial, sans-serif; } Text styling
